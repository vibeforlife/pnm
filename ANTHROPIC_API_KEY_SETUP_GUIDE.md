# How to Get Your Anthropic API Key for Poker Coach

## Step-by-Step Guide

### 1. Go to Anthropic Console
Visit: **https://console.anthropic.com/**

### 2. Sign Up or Log In
- If you don't have an account, click "Sign Up"
- Use your email to create an account
- Verify your email address

### 3. Navigate to API Keys
- Once logged in, look for "API Keys" in the left sidebar
- Or go directly to: https://console.anthropic.com/settings/keys

### 4. Create a New API Key
- Click "Create Key" or "+ New Key"
- Give it a name (e.g., "Poker Night Manager")
- Click "Create"

### 5. Copy Your API Key
- **IMPORTANT:** Copy the key immediately!
- It looks like: `sk-ant-api03-...`
- You won't be able to see it again after closing the dialog
- Store it somewhere safe (password manager recommended)

## Troubleshooting

### "API Key Invalid" Error:
- Make sure you copied the entire key (starts with `sk-ant-`)
- Check for extra spaces at beginning/end
- Verify key hasn't been deleted in console
- Try creating a new key

### "Rate Limit Exceeded":
- Wait 1 minute and try again
- Unlikely in poker games (50/min limit)

### "Insufficient Credits":
- Add payment method in Anthropic Console
- Or wait for next month (free tier resets)

### Analysis Not Working:
- Check internet connection
- Make sure photo is clear and shows cards
- Try a different photo
- Check API key is saved correctly
